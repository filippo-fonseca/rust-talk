# Rust Talk

A terminal-based chat app built in Rust.

## Setup

**1. Clone the repo**

```
git clone git@github.com:filippo-fonseca/rust-talk.git
```

**2. Run**

To run the app, use:

```
cargo run
```

To create a usable version, run:

```
cargo build
```
